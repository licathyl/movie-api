Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

API Endpoints

Request URL HTTP Method Request body data format Response body data format
Get a list of all movies /movies GET None A JSON object holding data about all the movies: 
{
  Title: "Parasite",
  Description: “The struggling Kim family sees an opportunity...", 
  Genre: { 
    Name: "Thriller", 
    Description: "Thriller film, also known as suspense film..." 
  }, 
  Director: { 
    Name: "Bong Joon Ho", 
    Bio: "Bong Joon Ho is a South Korean film director, producer and screenwriter.", 
    Birth: "1969" 
  }, 
  ImagePath: "parasite.png", 
  Featured: true 
}
Get data about a single movie by title /movies/[Title] GET None A JSON object holding data about a single movie, containing a name, description, genre, and image URL: 
{ 
  Title: "Guardians of the Galaxy", 
  Description: "Peter escapes from the planet Morag...”, 
  Genre: { 
    Name: "Action", 
    Description: "Action film is a film genre in which the protagonist..." 
  }, 
  Director: { 
    Name: "James Gunn", 
    Bio: "James Francis Gunn Jr. is an American film director, actor, screenwriter and former musician.", 
    Birth: "1966" 
  }, 
  ImagePath: "guardiansofthegalaxy", 
  Featured: false 
}
Get data about a genre by name/title /genres/[Name] GET None A JSON object holding data about a genre, like a description: 
{ 
  Name: "Action", 
  Description: "Action film is a film genre in which the protagonist..." 
}
Get data about a director by name /directors/[Name] GET None A JSON object holding data about a director, containing a bio, birth year, and death year, structured like: 
  
{ 
  Name: "Bong Joon Ho", 
  Bio: "Bong Joon Ho is a South Korean film director, producer and screenwriter.", 
  Birth: "1969" 
}
Add a user /users POST A JSON object holding data about the user to add, structured like: 
{ 
  Username: "cathy", 
  Password: "test123", 
  Email: "cathy@email.com", 
  Birthday: "2000-02-02",
  FavoriteMovies: [] 
}
A JSON object holding data about the user that was added, including an ID: 
{ 
  id : 1, 
  Username: "cathy", 
  Password: "test123", 
  Email: "cathy@email.com", 
  Birthday: "2000-02-02",
  FavoriteMovies: []
}
Update a user's information /users/[Username] PUT None A text message indicating the username, password, email and date of birth that was updated
Add a movie to list of favorites /users/[Username]/movies/[MovieID] POST A JSON object holding data about the movie to add, structured like: 
{ 
  Title: "Legally Blonde", 
  Description: "Elle's boyfriend breaks up with her...", 
  Genre: { 
    Name: "Romance", 
    Description: "Romance films make the romantic love..." 
  }, 
  Director: { 
    Name: "Robert Luketic", 
    Bio: "Robert Luketic is an Australian film director and writer.", 
    Birth: "1973" 
  }, 
  ImagePath: "legallyblonde.png", 
  Featured: false 
}
A text message indicating whether the movie was successfully added
Remove a movie from list of favorites /users/[Username]/movies/[MovieID] DELETE None A text message indicating whether the movie was successfully removed
Deregister a user /users/[Username] DELETE None A text message indicating whether the user was successfully removed
